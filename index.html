Características

Velas Japonesas en Vivo

Datos de mercado cripto en tiempo real vía WebSocket de Binance, sin delays. Múltiples intervalos de 1m a 1w.

Perfil de Volumen

Perfil de Volumen de Rango Variable con cálculo de POC, VAH y VAL.

Indicadores Técnicos

SMI (Squeeze Momentum), ADX con key level, medias móviles 50/100/200 y más. Calculados en cliente, sin servidores.

Totalmente Interactivo

Paneo y zoom en ambos ejes, crosshair, tooltip contextual y squeeze dots. Optimizado para escritorio y móvil.

Sin Registro

Sin cuentas, sin costos, sin anuncios. Abre el gráfico y empieza a analizar. Gratis para siempre.

Open Source

Código abierto en GitHub. Construido con React y Canvas API para máximo rendimiento.

Resposive y Touch

Diseñado para funcionar en escritorio, tablet y móvil. Gestos táctiles con pinch zoom y pan con un dedo.

Indicadores Disponibles

  • PVRV Perfil de Volumen de Rango Variable
  • Medias Móviles SMA 50, 100, 200
  • SMI Squeeze Momentum Indicator
  • ADX Average Directional Index con key level 23
  • +DI / -DI Direccional positivo / negativo
  • Squeeze Dots Detección de volatilidad comprimida
  • Crosshair Crosshair sincronizado entre paneles

Acerca de OpenCharts

OpenCharts es una plataforma de código abierto para análisis técnico de criptomonedas. Nuestro objetivo es proporcionar herramientas profesionales de trading accesibles para todos, sin barreras de entrada.

El proyecto consume datos públicos de la API de Binance y los procesa íntegramente en el navegador del usuario. No almacenamos datos de usuario, ni requerimos registro, ni mostramos publicidad intrusiva.

Toda la lógica de cálculo (SMI, ADX, medias móviles) se ejecuta en el cliente, garantizando transparencia y privacidad.

Privacidad

OpenCharts no recopila datos personales. No utiliza cookies de seguimiento ni servicios de analítica de terceros. El único almacenamiento local es la configuración de paneo/zoom del usuario.

Las únicas conexiones externas son a la API de Binance para obtener datos de mercado públicos. No se envía ninguna información del usuario a ningún servidor.

El códdigo fuente está disponible en GitHub para auditoría.
